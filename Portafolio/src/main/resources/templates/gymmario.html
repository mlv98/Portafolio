Gym-Fitness

Sitio web corporativo responsive desarrollado en WordPress con enfoque en captación de clientes y gestión de contenido.

Ver en GitHub

Descripción del Proyecto

Desarrollé esta solución utilizando un CMS (WordPress) para responder a una necesidad de negocio prioritaria: establecer una identidad digital rápida y escalable. El proyecto se centró en la experiencia de usuario (UX), permitiendo a los clientes consultar horarios, servicios y planes de entrenamiento de forma intuitiva. Aunque mi perfil es técnico (Java/Backend), este proyecto demuestra mi versatilidad para elegir la herramienta más eficiente según el problema (Time-to-market), mi capacidad de adaptación a entornos CMS y mis habilidades en diseño Frontend y maquetación.

Tecnologías usadas:
  • CMS: WordPress (Gestión de contenido).
  • CSS3 (Personalización de estilos), JavaScript.
  • Plugins/Tools: Elementor (Maquetación visual), Contact Form 7 (Gestión de leads)
