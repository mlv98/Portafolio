Gym-Fitness

Sitio web corporativo responsive desarrollado en WordPress con enfoque en captación de clientes y gestión de contenido.

Ver en GitHub

Descripción del Proyecto

Es una plataforma web de gestión para centros de entrenamiento que permite a los usuarios consultar horarios en tiempo real, visualizar planes detallados y explorar el catálogo de servicios. Desarrollada sobre WordPress para optimizar el tiempo, cuenta con una arquitectura escalable, diseño centrado en la experiencia de usuario y maquetación personalizada orientada a la conversión y retención de clientes

Tecnologías usadas:
  • Core: WordPress, PHP
  • CSS3
  • JavaScript
  • Plugins/Tools: Elementor, Contact Form, Lightbox, SlickNav
