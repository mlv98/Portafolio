Rampage

Aplicación de consulta y análisis de videojuegos, con integración de APIs externas, desarrollada en Spring Boot, Thymeleaf y MySQL.

Ver en GitHub

Descripción del Proyecto

Desarrollo Full Stack de una plataforma integral para gamers bajo arquitectura MVC. La funcionalidad principal es un motor de compatibilidad de hardware que, mediante un algoritmo propio, parsea requisitos de la API de RAWG y los compara matemáticamente con una base de datos local de benchmarks para validar el rendimiento del PC. El sistema implementa seguridad robusta (Spring Security con cifrado BCrypt y protección CSRF) , consumo de APIs externas mediante RestTemplate y WebClient , y un diseño responsive moderno con Tailwind CSS.

Tecnologías usadas:
  • Java 17
  • Spring Boot 3
  • Spring Security 6
  • MySQL 8
  • Thymeleaf
  • Tailwind CSS
  • JavaScript / Fetch API
