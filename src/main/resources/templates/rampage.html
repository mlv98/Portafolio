Rampage

Aplicación de consulta y análisis de videojuegos, con integración de APIs externas, desarrollada en Spring Boot, Thymeleaf y MySQL.

Ver en GitHub

Descripción del Proyecto

Rampage es una plataforma donde los usuarios pueden consultar información sobre videojuegos, analizar compatibilidad de hardware y seguir las últimas tendencias del mercado. Incluye integración con APIs externas como RAWG y NewsAPI, funcionalidades de filtrado y búsqueda avanzada, y un diseño full-stack en Spring Boot con persistencia en MySQL.

Tecnologías usadas:
  • Spring Boot
  • Thymeleaf
  • MySQL
  • JavaScript / AJAX
  • APIs externas (RAWG, NewsAPI)
