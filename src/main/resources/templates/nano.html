Nano-TI

Es un agente conversacional inteligente que elimina la burocracia en las solicitudes de material informático. El usuario interactúa con el bot a través de una interfaz web, y el sistema se encarga de validar los datos, solicitar aprobación al responsable y actualizar el inventario automáticamente..

Descripción del Proyecto

Nano TI es una solución tecnológica Low-Code diseñada para automatizar y optimizar el proceso de solicitud de material informático (hardware y software) en entornos empresariales. El proyecto nace de la necesidad de eliminar la burocracia manual, sustituyendo los correos electrónicos desestructurados por un Agente Conversacional Inteligente capaz de gestionar el ciclo de vida completo de una compra, desde la petición inicial hasta su aprobación final. La arquitectura del sistema se basa en la integración nativa del ecosistema Microsoft Power Platform. La interfaz principal ha sido desarrollada con Microsoft Copilot Studio, utilizando Inteligencia Artificial Generativa para interpretar el lenguaje natural del usuario, extraer entidades clave (como el tipo de material, coste estimado y justificación) y validar la identidad del solicitante mediante la captura de credenciales. Una vez recolectados los datos, la lógica de negocio es orquestada por Power Automate. Este servicio conecta el frontend con el backend, ejecutando un flujo en la nube que registra la solicitud en una base de datos relacional segura en Microsoft Dataverse. Simultáneamente, el sistema dispara un proceso de aprobación dinámico enviando una Tarjeta Adaptativa (Adaptive Card) a través de Outlook al responsable del departamento. Esto permite al gestor aprobar o rechazar la solicitud directamente desde su correo, sin necesidad de acceder a plataformas externas. Finalmente, el sistema cierra el ciclo actualizando en tiempo real el estado de la solicitud en Dataverse. Se ha implementado una lógica visual avanzada que modifica la columna de "Estado de Aprobación".

Tecnologías usadas:
  • Microsoft Copilot Studio: Desarrollo del agente conversacional, gestión de temas (Topics) e implementación de IA Generativa para la comprensión de intenciones.
  • Microsoft Power Automate: Diseño de flujos de nube (Cloud Flows) para la orquestación de datos, lógica condicional y gestión de aprobaciones automatizadas.
  • Microsoft Dataverse: Infraestructura de base de datos para la persistencia segura de las solicitudes, con configuración de columnas personalizadas y lógica de visualización.
  • Microsoft Outlook 365: Canal de comunicación para la recepción de notificaciones y gestión de tarjetas de aprobación interactivas.
